# SUNO Prompt Generator

Web アプリで簡単に SUNO で使えるプロンプトを生成します。

## 使い方
1. `index.html` をブラウザで開く（ローカルでも動作します）
2. 各項目を選択／入力して `Generate` をクリック
3. 生成されたプロンプトを `Copy` ボタンでクリップボードにコピー

## 技術スタック
- HTML5 + Tailwind CSS CDN
- Vanilla JavaScript (ES6)
